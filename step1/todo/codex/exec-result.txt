  테스트

  - 실행: 브라우저로 index.html 열기 → 콘솔에 PASS/FAIL/요약 출력
  - 커버리지:
      - Todo 항목 추가 유효성 검사 및 정상 생성
      - 완료 상태 토글
      - 상태 필터링 로직
      - 로컬 스토리지 저장/불러오기
      - Store의 추가/삭제/토글 동작
      - 필터와 검색 조합 동작
  - 메모리 스토리지를 이용해 스토리지 의존성 분리로 테스트 독립성 확보

  사용 방법

  - 할 일을 입력하고 우선순위/마감일 지정 후 “추가”
  - 체크박스로 완료 토글, “수정”으로 항목 편집, “삭제”로 제거
  - 상단 필터(전체/미완료/완료) 및 검색 사용
  - 카운트: 전체/미완료/완료 수 실시간 표시
  - 데이터는 자동으로 로컬 스토리지에 저장/복원
